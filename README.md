# NVoice
P2P Voice с расширенными возможностями

# Как пользоваться:
1. Запуск Nvoice.exe
2. Войти в сеть
3. Поделиться подключением
4. Отправить файл-приглашение другу
5. Добавить подключение друга
6. Дождаться пока статус сменится на мигающий красный (ожидание)
7. Можно подключаться, появится статус желтный (готово к воспроизведению)
8. Подождать пока друг нажмёт подключиться и сменится статсу на зелёный (работает)
9. Можно общаться!

# Как настроить стиль под себя
В файле style.qss можно менять любые стили приложения, стили для QWidget считается применимым к большинству объектов в приложении.

# Возможные ошибки
- Не получен внешний IP (в поле IP адрес останется надпись XX.XX.XX.XX)
Возможное решение: создать файл StunServers рядом с программой (NVoice.exe) и записать в него какой-либо рабочий/доверенный STUN сервер.
